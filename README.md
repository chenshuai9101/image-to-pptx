# Image to PPTX 🖼️

> 图片批量转PPT工具 — 自动排版生成演示文稿

[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

## 概述

将图片批量转换为专业PPT演示文稿，支持自动排版、尺寸适配和批量处理。

## 核心功能

- **📸 批量导入** — 支持文件夹批量导入图片
- **📐 智能排版** — 自动适配图片尺寸和PPT版式
- **🎨 模板支持** — 多种演示模板可选
- **⚡ 快速导出** — 一键生成 .pptx 文件
